PCB ReTrace

Digitize, document, and reverse engineer printed circuit boards. Map components, trace nets, inspect layers, and generate schematics. Runs entirely in your browser.

Privacy First

Zero cloud upload. All images and data are stored in your browser's local database. Works 100% offline.

Auto-Stitch & X-Ray

Align close-up shots with wide-angle views. Stitch Top vs. Bottom layers to inspect traces through the board.

📝

Visual BOM

Map components directly on the image. Record values, references, and descriptions, then export to CSV.

🕸️

Net Tracing

Document connectivity across layers. Define nets, label nodes, and export netlists (.net,.cir for KiCad, SPICE).

Schematics

Generate interactive schematic diagrams, resolve placeholders into real components, and export to KiCad (.kicad_sch).

🔗

Deep Linking

Share projects instantly. Host your Device ZIP on GitHub or any CORS-enabled server and open it via a direct URL.
